# Event App


## Project setup

Please ask the  team for the .env file. Without it no connection to Airtable can be established.

```sh
npm install
```

### Compiles and hot-reloads for development

```sh
npm run serve
```

### Compiles and minifies for production

```sh
npm run build
```

### Lints and fixes files

```sh
npm run lint
```
